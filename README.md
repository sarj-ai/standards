# sarj-ai/linting

Lint rules + shared strict configs for TypeScript + Python.

| Package | Tool | Registry |
|---|---|---|
| [`@sarj/eslint-plugin`](packages/eslint-plugin/) | ESLint | [npm](https://www.npmjs.com/package/@sarj/eslint-plugin) |
| [`sarj-python-lint`](packages/python-lint/) | pre-commit (Python + SQL) | [PyPI](https://pypi.org/project/sarj-python-lint/) |
| [`sarj-lint-configs`](packages/lint-configs/) | strict ruff / pyright / ESLint configs | [PyPI](https://pypi.org/project/sarj-lint-configs/) |
| [`ast-grep-rules/`](ast-grep-rules/) | 6 declarative YAML rules for `ast-grep scan` | — |

Each rule is self-documenting via its source file. See per-package READMEs for the rule index and adoption instructions. MIT.
